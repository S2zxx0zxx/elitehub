## Key Decisions
- Standardized color system based on MyJio UI (Elite White, Brand Yellow, near-black dark mode)
- Mobile-first bottom navigation paradigm
- Server Actions for robust form submissions
- Direct-to-R2 upload bypasses Next.js server limits
- Next.js App Router selected
- Auth migrated fully to Clerk (lazy sync with Neon DB via `getDbUser()`)
- R2 signed URLs serve all media to prevent exposing raw mediaKey. Private posts are guarded by a server-side purchase/subscription check.
